
# Netflix Clone

This project is complete clone of Netflix. It was created with Kotlin. It uses The [TheMovieDb](https://www.themoviedb.org/) Api to search for movies and display details. Feel free to contribute!


## Tools used


 * Android
 * FirebaseFirestore (Backend-as-a-Service (BaaS))
 * Volley library
 * Kotlin
 * XML

## Local setup

* Generate an api_key from https://developers.themoviedb.org/3
* Update the following line in local.properties

 * [api_key=YOUR_TMDB_API_KEY]
